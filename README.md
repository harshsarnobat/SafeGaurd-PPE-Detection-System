# 🦺 SafeGaurd

SafeGaurd is an AI-powered workplace safety monitoring system that uses Computer Vision and Deep Learning to detect safety violations in real time. It helps improve workplace safety by identifying helmet violations, restricted area intrusions, and mobile phone usage, while providing instant audio alerts and maintaining incident logs.

---

## 🚀 Features

- 🪖 Real-time Helmet Detection
- 🚧 Restricted Area Intrusion Detection
- 📱 Mobile Phone Usage Detection
- 🔊 Instant Audio Alerts
- 📝 Automatic Incident Logging
- 🎥 Live Video Monitoring
- ⚡ Fast and Efficient Object Detection

---

## 🛠️ Tech Stack

- Python
- YOLOv8
- OpenCV
- Ultralytics
- NumPy
- CVZone
